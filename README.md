# Mini-Compiler-With-GUI

Mini Compiler Built Using Java (Compiler Construction Project). It has enormous number of fuctionalities like: 
1. Symbol Table
2. View Tokens
3. Formatting
4. Intermediate Code Generation (3-Address Code)
